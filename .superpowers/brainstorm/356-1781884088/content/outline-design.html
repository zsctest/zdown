// 数据结构
struct OutlineItem { level: u8, text: String, line: usize }

// 从 AST 提取
fn extract_outline(doc: &Document) -> Vec<OutlineItem>
→ 遍历 doc.blocks，筛选 Block::Heading
→ 用 inlines_to_plain 提取纯文本
→ 用 span.start_line 获取行号

// 渲染面板

→ egui::ScrollArea 包含可点击标题
→ 缩进 = (level - 1) * 16px
→ H1 粗体，H3+ 小号字体
→ 点击 → editor.set_cursor(Cursor::new(line, 0))

2. UI 集成（main.rs）

在现有视图外部包裹 SidePanel::left，不改变内部视图逻辑。

// main.rs 修改（伪代码）
fn ui(&mut self, ui: &mut egui::Ui, ...) {
    menu::show_menu(...);
    // ↓ 新增
    egui::SidePanel::left("outline_panel")
        .resizable(true)
        .default_width(200.0)
        .show_inside(ui, |ui| {
            outline_view::show_outline_panel(ui, &mut self.state);
        });
    // ↓ 原有的中央区域
    egui::CentralPanel::default().show_inside(ui, |ui| {
        match self.view_mode { ... }  // 源码/预览/Hybrid
    });
}

4. 边界情况

场景行为
空文档面板显示 "（无标题）"
文档只有正文无标题同上
标题文本为空（###）显示为 "(空标题)"
标题文本很长CSS text-overflow:ellipsis，悬停 tooltip
语法错误的 #（不构成标题）parser 不识别为 Heading，不出现在大纲
文档编辑中每帧重新解析，大纲实时更新
点击不存在的行号set_cursor 验证行号有效性，panic-free
面板宽度为 0（用户拖拽关闭）正常，可拖回
