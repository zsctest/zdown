哪些视图需要搜索功能？

zdown 有三种视图模式：源码、预览、Hybrid。搜索功能应在哪些视图中可用？

源码视图 (Ctrl+1)
1 # 我的文档
2
3 这是搜索文本
4
5 又一段搜索内容

源码视图

纯文本 Markdown 源码，搜索+高亮匹配最直观

✅ 必须支持

预览视图 (Ctrl+2)

我的文档

这是搜索文本

又一段搜索内容

预览视图

渲染后的 Markdown，匹配的是显示文本而非源码

可选，实现更复杂

Hybrid 视图 (Ctrl+3)

我的文档

这是搜索文本

又一段搜索内容

Hybrid 视图

光标所在 block 显示源码，其余渲染显示。搜索可能需要在两种渲染方式间切换

可选，较复杂

A

仅源码视图

只在源码模式(Ctrl+1)下 Ctrl+F 可用。如果用户在预览/Hybrid 模式按 Ctrl+F，自动切换到源码视图。

B

所有视图（推荐）

先在源码视图中实现搜索+高亮替换。预览/Hybrid 视图中的搜索可以后续迭代添加（搜索时 UI 切换到源码模式）。
