搜索栏放在哪里？

Ctrl+F 激活搜索后，搜索栏应出现在编辑器的哪个位置？

A

顶部浮动栏（推荐）

搜索栏以浮动条形式出现在编辑器顶部，不占用代码编辑区域。类似 VS Code / Sublime Text 的搜索体验。可以用 Esc 关闭。

B

底部状态栏内嵌

搜索栏嵌入在编辑器底部的状态栏区域，与现有的导出状态栏共存。空间紧凑，但可能显得拥挤。

C

弹出对话框

使用 egui 的 Window 组件弹出一个独立的搜索对话框。完全不影响编辑器布局，但会遮挡部分内容。
